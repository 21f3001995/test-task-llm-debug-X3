# Project Title: Sales Dashboard

## Brief
This project is a dynamic sales dashboard that allows users to view product sales and convert total sales amounts into different currencies. The dashboard includes a product table, a currency selector, and displays the converted total sales dynamically. The application also logs conversion rates and events to the console for debugging purposes.

### Features
1. **Product Table**: Displays a list of products and their sales data.
2. **Currency Selector**: Allows users to select a currency for conversion using data from `rates.json`.
3. **Dynamic Total Sales**: Shows the converted total sales amount in the `#total-sales` section.
4. **Console Logging**: Logs conversion rates and relevant events to the console for tracking.

## Deployment
To deploy this project, follow these steps:
1. Clone the repository: `git clone https://github.com/yourusername/sales-dashboard.git`
2. Navigate to the project directory: `cd sales-dashboard`
3. Open `index.html` in your web browser to view the dashboard.
